This project contains a Stripe Payment Integration setup with two folders:

1. client  (Frontend)


2. server  (Backend)






How to Run the Project

1. Download the ZIP file and extract it.


2. Open two separate terminals: one for the client folder and one for the server folder.



Client Terminal: cd client

Server Terminal: cd server




Installing Dependencies

Client Side: cd client
npm install

If the project does not run, install this package: npm install react-router-dom

Server Side: cd server
npm install

client start command is   npm run dev

and install nodemon to run server
npm instal -g nodemon

server start command is nodemon

If some dependencies are still missing, run: npm install body-parser cors dotenv express mongoose stripe




Notes

The project will not work without the .env file on the server side.
Contact me to get the .env file.

After installing dependencies, run the client and server normally using: For Client: npm run dev
For Server: node app.js   or use nodemon
